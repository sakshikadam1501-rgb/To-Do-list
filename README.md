# To-Do-list

📋 To-Do List CLI Application

A simple command-line task management tool that allows users to add, complete, and delete tasks.
Tasks are saved in a file to ensure persistence between program runs.

✨ Features

• Add new tasks
• Mark tasks as completed
• Delete tasks
• Task storage using JSON file
• Menu-driven interface
• Colored terminal output

▶ Run

python todo_app/todo.py
